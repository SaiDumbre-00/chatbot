# 🤖 Python Chatbot

A simple and smart rule-based chatbot built using Python.  
It uses pattern matching and predefined intents to simulate human-like conversation.

---

## 🚀 Features
- Pattern matching using keywords  
- Multiple intents (greeting, help, jokes, etc.)  
- Remembers user name  
- Displays current time  
- Basic knowledge base  
- Random responses  
- Conversation logging  

---

## 🛠️ Technologies Used
- Python 3  
- Built-in libraries: `random`, `datetime`
